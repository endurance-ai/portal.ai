## 1. Daydream 대비 MOODFIT 포지셔닝 요약

```
Daydream = "AI Fashion Chatbot"     →  텍스트 대화로 쇼핑
MOODFIT  = "AI Look Analyzer"       →  이미지 한 장으로 스타일 해부
```

| 차원 | Daydream | MOODFIT | 우위 |
|------|----------|---------|------|
| **진입 장벽** | 온보딩 5~6단계 | 이미지 업로드 1단계 | MOODFIT |
| **핵심 경험** | 챗 → 상품 리스트 | 이미지 → 핫스팟 분해 → 상품 매칭 | MOODFIT (시각적 차별화) |
| **결과 리파인** | "Say More" (대화형) | ❌ 없음 | Daydream |
| **개인화** | Style Passport (진화형) | ❌ 없음 (단발성) | Daydream |
| **가격 필터** | 온보딩에서 수집 + "더 저렴하게" | ❌ 없음 | Daydream |
| **브랜드 커버리지** | 10,000+ 브랜드, 2M 상품 | SerpApi 실시간 (제한적) | Daydream |
| **플랫폼** | iOS 앱 전용 | 웹 (URL 공유 가능) | MOODFIT |
| **시장** | 미국/글로벌 (영어) | 한국 + 글로벌 | MOODFIT (한국 시장) |
| **수익** | CPA (커머스 수수료) | 프리미엄 구독 + 어필리에이트 | 미정 |

### MOODFIT이 지켜야 할 차별점
1. **이미지 → 룩 분해** (핫스팟 + 커넥터 라인) — Daydream에 없는 핵심 UX
2. **크로스 가격대** (₩3만~₩300만 동시 제안) — Daydream은 럭셔리 편향
3. **웹 퍼스트** — URL 공유 → 바이럴 잠재력
4. **한국 브랜드 커버리지** — 무신사, W컨셉, 29CM

### MOODFIT이 메워야 할 격차
1. 가격 필터링 (현재 없음 → 즉시 추가 필요)
2. 결과 리파인먼트 (MVP에서 추가 고려)
3. 개인화/학습 루프 (MVP~Beta)
4. 상품 DB 커버리지 (SerpApi → 어필리에이트 전환)

## 2. 영역별 핵심 발견 요약

### 2-A. 프론트엔드 입력 설계

**발견**: 업계가 두 갈래로 분화 중
- **대화형** (Daydream, Google AI Mode) — 필터 대신 자연어
- **구조화 필터** (무신사, 지그재그, 에이블리) — 칩/슬라이더/탭

**MOODFIT 권장**: 하이브리드
- 업로드 시: 가격대 칩 (💰/💰💰/💰💰💰) + 성별 확인
- 결과 후: "더 저렴하게" / "더 캐주얼하게" 칩 (Say More 라이트 버전)

**성별 트렌드**: 이진 분류 → 체형/선호 기반으로 이동 중. MOODFIT의 AI 추론 + 유저 수정 방식이 트렌드에 부합.

### 2-B. 상품 DB 구축

**핵심 발견**:

| 채널 | MOODFIT 적합도 | 블로커 |
|------|---------------|--------|
| **SerpApi** (현재) | POC에 적합 | 월 100회 한도, 수익 연결 불가 |
| **쿠팡 파트너스** | MVP 1순위 | 15만원 매출 후 API 접근, **시간당 10회 호출 제한** |
| **LinkPrice** | MVP 핵심 | ~500 가맹점, 딥링크 API, 한국 패션 브랜드 커버리지 |
| **H&M Affiliate** | SPA 브랜드 대응 | 7~11% 수수료, COS/ARKET 포함 — **API 접근 가능** |
| **AWIN** | 글로벌 확장 시 | 30,000+ 브랜드 (Nike, ASOS, COS) |
| **무신사** | API 없음 | 큐레이터 프로그램만 존재 (SNS/인플루언서 기반) |

**법적 발견**: 대법원 2021도1533 판결 — 접근 제어 우회 없는 크롤링은 무죄. 그러나 어필리에이트 API가 법적 리스크 제로이므로 우선.

**벡터 DB**: POC는 Qdrant Docker (무료), 한국어 임베딩은 `bge-m3-korean`이 `text-embedding-3-small`보다 우수하나 POC에서는 OpenAI로 충분.

### 2-C. 피드백 파이프라인

**핵심 발견**: 패션 도메인에서 **implicit feedback이 압도적으로 우월**.

**권장 3단계 수집**:
1. **Always-visible**: 👍/👎 + 저장 버튼 (모든 상품 카드)
2. **Session-end**: 전체 추천 만족도 슬라이더 (선택적)
3. **Periodic**: 스타일 이미지 평가 마이크로 설문 (Pinterest 방식)

**피드백 → 벡터 업데이트 방식**: 가중 평균 + 시간 감쇠(exponential decay), 5분 미니배치로 Qdrant upsert.

**추천 메트릭**: Product CTR, Save Rate, NDCG@4, 커스텀 **Style Relevance Score (SRS)**.

**A/B 테스트**: PostHog (무료 1M 이벤트/월) + Vercel Edge Config.

## 5. 비용 임팩트 정리

### POC (현재)
```
AI 분석:        ~₩6/건 (GPT-4o-mini)
상품 검색:      SerpApi 무료 100회/월
서버:           Vercel 무료
DB:             없음
────────────────────────
월 예상:        ~₩10,000
```

### MVP (목표)
```
AI 분석:        ~₩6/건
상품 검색:      LinkPrice 무료 (어필리에이트)
벡터 DB:        Qdrant Docker (무료) or Cloud ($25/월)
서버:           Vercel Pro ($20/월)
PostHog:        무료 (1M 이벤트)
────────────────────────
월 예상:        ~₩90,000 (100명 기준)
```

### 수익 전환점
```
쿠팡 파트너스:  3% CPA
LinkPrice:      가맹점별 상이 (5~15%)
H&M Affiliate:  7~11%

BEP: 70명 Basic(₩4,900) 구독자
     또는 월 ₩300만 어필리에이트 GMV (평균 8% 수수료 기준 ₩240K)
```
