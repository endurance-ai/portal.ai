## 3. 지원 내용 (5가지)

| # | 영역 | 상세 |
|---|------|------|
| 1 | **글로벌 진출** | 1인당 100만원 해외 진출 지원금 + Go-to-Market/UX 리서치 등 전문가 멘토링 |
| 2 | **사업 전환** | 선배 창업가 멘토링 + IR 고도화 + 팀별 전담 코치 밀착 지원 (계획→실행→회고) |
| 3 | **또래 커뮤니티** | 20대 동료 창업가 네트워크 + 피어 러닝 |
| 4 | **MARU 네트워크** | 500+ 스타트업 커뮤니티 (공간·성장·커뮤니티·투자·홍보) + **연간 ~13억원** 제휴 혜택 (클라우드·AI·개발·채용·홍보) |
| 5 | **데모데이** | 정주영 창업경진대회 본선 진출 기회 (결선 4팀, **총상금 7천만원**) |

## 4. 선발 기준 & 심사 요소

### 공식 기준
- **팀 역량** — 실행력, 보유 기술, 팀 구성 다양성
- **비즈니스 모델** — 문제 정의, 솔루션 차별성, 시장 규모, 수익 모델
- **기업가정신** — 도전 의지, 글로벌 마인드셋, 나눔(pay-it-forward)

### 에세이에서 읽히는 평가 포인트
| 문항 | 평가하는 것 |
|------|-------------|
| Q1 (아이디어) | 문제 정의 깊이, 솔루션 차별성, 고객 이해도 |
| Q2 (글로벌 경쟁력) | 시장 분석력, 경쟁 우위 논리, 진입 전략 구체성 |
| Q3 (역량·나눔) | 기술/역량 증명, 실적 기반 근거, **pay-it-forward 진정성** |
| Q4 (도전 사례) | 그릿(Grit), 문제 해결 프로세스, 학습-적용 루프 |
| Q5 (지원 동기) | 시의성("왜 지금"), 프로그램 이해도, **측정 가능한 목표** |

### 참가 자격 (적합한 사람)
- 글로벌 시장 경쟁 의지가 있는 자
- 창업에 몰입할 준비가 된 자
- 핵심 과정(팀빌딩, 제품 제작)을 직접 실행 가능한 개인/팀
- 글로벌 시장 리서치와 고객 인터뷰 수행 가능자

## 9. 에세이 작성 전략 가이드

### 공통 원칙
1. **500자 = 짧다** — 한 문항에 하나의 핵심 메시지만
2. **수치로 증명** — "65% 전환율", "230% 매출 성장" 같은 구체 숫자
3. **STAR 구조** — Situation → Task → Action → Result (+ Learning)
4. **재단 가치 정렬** — 도전(실행력) / 성장(학습 루프) / 나눔(pay-it-forward)
5. **Born Global** — 처음부터 글로벌을 전제한 사고

### 문항별 핵심 전략

#### Q3. 핵심 역량 + pay-it-forward (엔지니어 관점)
- **묻는 것**: "동료 창업가들과 나눌 수 있는" 역량 → 기술만 나열 X, **나눔 가능성** 강조
- **전략**: 기술 스택 나열보다 → "이 기술로 무엇을 만들었고, 동료에게 어떻게 도움이 되는가"
- **엔지니어 차별점**: AI/프로덕트 빌딩 능력은 대부분 팀에게 가장 부족한 리소스 → 구체적 기여 가능

#### Q4. 가장 힘들었던 사례
- **묻는 것**: 실패/도전에서 배운 것 → **그릿 + 학습 루프**
- **전략**: ① 왜 → ② 무엇을 → ③ 결과(수치) → ④ 이후 변화 (4단계 필수)
- **함정**: "힘들었지만 극복했다" 미담 X → 구체적으로 무엇이 바뀌었는지

#### Q5. 지원 동기 + 목표
- **묻는 것**: "왜 지금, 왜 나, 왜 두어스" + **측정 가능한 지표**
- **전략**: 개인 내러티브("패션 모델 → 엔지니어") + 프로그램 구체 활용 계획 + KPI
- **킬러 요소**: "정주영 데모데이 직관 경험" 같은 직접 접점 언급

### 차별화 포인트 (엔지니어)
- 대부분 지원자는 비즈니스/세일즈 백그라운드 → **제품을 직접 만드는 기술 창업가**는 희소
- AI 프로덕트를 0→1로 빌딩한 실적 (fashion-ai 서비스 자체)
- "기획만 하는 사람" vs "코드로 증명하는 사람" 대비 강조
- 동료 창업가에게 **기술 역량 나눔** (MVP 빌딩, AI 활용법, 프로덕트 아키텍처 리뷰 등)
