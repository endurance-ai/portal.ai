## 2. Daydream UI/UX 분석

### 디자인 철학
- **"Liquid Glass"** — iOS 26의 반투명/빛/부드러운 입체감 적극 활용
- **"유틸리티 + 비주얼 탐닉"** — 깔끔한 내비게이션 + 몰입형 상품 디스플레이
- 전통적 이커머스 UI가 아닌 **"스타일리스트와 대화하는 경험"** 지향

### 핵심 인터페이스

| 기능 | 설명 |
|------|------|
| **Chat-to-Shop** | 자연어 입력 → AI가 상품 추천 ("I need a dress for a beach party in Bali") |
| **이미지 검색** | 스크린샷/사진 업로드 → 유사 상품 매칭 |
| **Style Passport** | 유저 프로필이 진화 — 사이즈, 선호 브랜드, 가격대, 스타일 학습 |
| **Say More** | 결과 미세 조정 ("좀 더 밝은 색으로", "가격대 낮춰줘") |
| **자동 컬렉션** | 저장한 아이템이 자동으로 카테고리별 정리 |

### UX 플로우
```
온보딩 (이름, 생일, 가격대, 브랜드 선호)
  ↓
Style Passport 생성
  ↓
채팅 or 이미지 업로드
  ↓
AI 추천 결과 (실시간 브라우징 중 대화 가능)
  ↓
"Say More"로 미세 조정
  ↓
상품 클릭 → 외부 쇼핑몰로 리다이렉트 (자체 결제 없음)
```

### 약점
- **자체 결제 없음** — 외부 쇼핑몰로 리다이렉트
- **앱 전용** (iOS) — 웹 접근성 제한적
- **영어/미국 시장 집중** — 한국/아시아 브랜드 커버리지 약함
- **채팅 중심** — 비주얼 분석(룩 분해)보다 텍스트 대화에 치중

## 3. MOODFIT vs Daydream 비교

| 항목 | Daydream | MOODFIT |
|------|----------|---------|
| **입력 방식** | 텍스트 채팅 + 이미지 | **이미지 중심** (룩 분해) |
| **핵심 기능** | 대화형 쇼핑 에이전트 | **비주얼 룩 분석 + 아이템별 매칭** |
| **결과 형태** | 상품 목록 나열 | **원본 이미지 위 핫스팟 → 아이템별 추천** |
| **개인화** | Style Passport (대화 기반) | 이미지 기반 무드 분석 (시각 중심) |
| **시장** | 미국/글로벌 (영어) | **글로벌 + 아시아 브랜드** 커버리지 |
| **플랫폼** | iOS 앱 | **웹 우선** (접근성 ↑) |
| **브랜드** | 10,000+ (럭셔리 중심) | 크로스플랫폼 (럭셔리 ~ SPA) |
| **비용** | 무료 (커머스 수수료) | 프리미엄 + 어필리에이트 |

## 4. MOODFIT 차별화 전략

### A. "Look Decomposition" — 룩 분해 비주얼라이제이션

Daydream은 이미지를 넣으면 **비슷한 상품 목록**을 보여줌.
MOODFIT은 이미지를 넣으면 **룩 전체를 분해**해서 각 아이템에 핫스팟 + 커넥터 라인으로 연결.

→ **"인터랙티브 룩북"** 경험. 패션 매거진 에디토리얼 느낌.

이건 Daydream에 없는 기능이고, 시각적 임팩트가 커서 **바이럴 가능성**이 높음.

### B. 크로스 가격대 매칭

Daydream은 럭셔리 중심 (Net-A-Porter, SSENSE).
MOODFIT은 **같은 스타일을 $50 ~ $5,000까지** 가격대별로 보여줌.

```
"이 THE ROW 코트 스타일을 UNIQLO 가격에서도 찾아줄게"
```

→ 가격 민감한 유저 (학생, 20대) 공략. Daydream과 타겟 분리.

### C. Mood DNA — 스타일 분석 리포트

단순 상품 추천이 아닌 **무드 분석 데이터** 제공:
- 무드 태그 + confidence score
- 추출 컬러 팔레트
- 스타일 요약 (에디토리얼 톤)

→ 유저가 **자기 스타일을 "이해"**하게 되는 경험. SNS 공유 유도.

### D. 웹 퍼스트 + 글로벌

Daydream은 iOS 앱 전용.
MOODFIT은 **웹 우선** → 링크 하나로 접근, 앱 설치 불필요.

→ 바이럴에 유리. "이 사이트에 사진 넣어봐" 한 마디로 공유 가능.

### E. 아시아 브랜드 커버리지

Daydream은 서양 럭셔리 중심.
MOODFIT은 **무신사, W컨셉, 29CM, ZARA, COS** 등 한국/아시아 브랜드 포함.

→ 한국 시장 + 해외 K-fashion 관심 유저 동시 공략.

## 5. MOODFIT 로드맵 방향

### Phase 1: POC (현재)
```
✅ 이미지 → 룩 분해 + 무드 분석
⬜ 실제 상품 검색 API 연동
⬜ Mood DNA 리포트 공유 기능 (SNS 바이럴)
```

### Phase 2: MVP
```
⬜ 크로스플랫폼 상품 검색 (어필리에이트 API)
⬜ 가격대 필터 ($, $$, $$$)
⬜ 유저 히스토리 → 스타일 프로필 (Daydream의 Style Passport 대응)
⬜ "이 스타일 더 보기" → 연관 룩 추천
```

### Phase 3: Growth
```
⬜ 모바일 앱 (PWA 또는 네이티브)
⬜ 브랜드 파트너십 (공식 API 연동)
⬜ 커뮤니티 — 유저가 올린 룩 갤러리
⬜ 다국어 (EN, KO, JP, ZH)
```

## 6. 핵심 한 줄 포지셔닝

```
Daydream = "AI 패션 챗봇" (대화로 쇼핑)
MOODFIT  = "AI 룩 분석기" (사진으로 스타일 해부)
```

**Daydream이 "무엇을 살까"에 답한다면,
MOODFIT은 "이 스타일이 뭔데? 어디서 살 수 있어?"에 답한다.**
