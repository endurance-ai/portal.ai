# archive

종료된 산출물 보관소. **개발 작업 시 이 디렉터리는 탐색·참조하지 않는다.**

git history 보존을 위해 이동만 하고 삭제하지 않음. 회고나 지원서 작성 시 들춰볼 용도.

## 구조

| 디렉터리 | 내용 |
|----------|------|
| `handoffs/` | 종료된 세션 인수인계 (HANDOFF*.md) |
| `plans/` | 머지 완료 또는 폐기된 plan 문서 |
| `superpowers-plans/` | superpowers 스킬로 생성된 sprint plan (전부 종료) |
| `specs/` | 종료된 디자인/설계 스펙 |
| `research/` | 1회성 결정 근거 리서치 (계속 보는 reference는 `docs/research/` 에 잔류) |
| `eval/` | v4 시점 평가 산출물 — v5에서 새로 작성 예정 |
| `presentation/` | 발표/장표 자료 |
| `DESIGN.legacy.md` | Digital Atelier (구 디자인 시스템). 현재는 root `DESIGN.md` 활성 |
