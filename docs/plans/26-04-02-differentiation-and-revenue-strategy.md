## 1. 시장 현황: 왜 다 죽었는가

### 한국 크로스플랫폼 패션 서비스 흥망사

한국에서 "여러 쇼핑몰 상품을 모아서 보여주는" 패션 서비스는 수십 개가 시도되었고, **대부분 죽었다.**

#### 사망 목록

| 서비스 | 기간 | 모델 | 전성기 규모 | 사망 원인 |
|--------|------|------|-----------|----------|
| **캐치패션** | 2019-2024 | 럭셔리 메타서치 (40+ 글로벌 파트너 API) | 1.5만 브랜드, 400만 상품 | 매출 40억 vs 영업손실 69억. 커미션이 CAC를 못 감당. 쿠팡의 Farfetch 인수로 투자 위축 |
| **발란** | 2015-2026 | 럭셔리 마켓플레이스 | 누적 적자 7,840억 | **2026년 2월 파산 선고**. 3사 합병 실패, 셀러 정산 지연, Series D 무산 |
| **스타일쉐어** | 2011-2022 | 패션 소셜커머스 | Gen Z 여성 커뮤니티 | 무신사에 100% 인수 후 서비스 종료 |

### 실패 공통 패턴 5가지

#### 패턴 1: "검색만 해주는 서비스"는 100% 죽는다
캐치패션이 완벽한 증거. 40개 글로벌 파트너, 400만 상품, 재고/물류 부담 제로 — **그래도 죽었다.** 순수 어필리에이트 커미션(3-7%)으로는 고객 획득 비용을 영원히 감당할 수 없다.

#### 패턴 2: 통합 결제 없으면 유저가 떠난다
지그재그 교훈. 메타서치에서 Z-Pay(통합 결제)로 전환한 뒤에야 생존. "상품 보여주고 외부로 보내기"만으로는 리텐션이 안 잡힌다.

#### 패턴 3: 커미션 10%면 셀러가 도망간다
지그재그가 10% 커미션 붙였더니 **2주 만에 500/1,000 셀러가 이탈**. 한국 패션 마진이 그만큼 박하다.

#### 패턴 4: 크롤링하면 고소당할 수 있다
캐치패션이 발란/트렌비/머스트잇을 크롤링 혐의로 **형사 고소**. 정보통신망법 48조 위반 소지. 법적 리스크가 실재한다.

#### 패턴 5: 대기업 없으면 생존 못 한다
2년간 20개+ 이커머스 폐업. 독립 플랫폼은 인수되거나 죽는다. 무신사/카카오/신세계/네이버 산하만 생존.

### 유일한 생존 조건 (3가지 중 1개 이상)

```
1. 통합 결제 보유 (지그재그 Z-Pay)
2. 날카로운 버티컬 집중 (룩핀 = 남성, 젠테 = 럭셔리 부티크)
3. 대기업 인수/백업 (29CM → 무신사, W컨셉 → 신세계)
```

## 3. portal.ai 차별점: 왜 우리가 다른가

### 3-1. AI 이미지 분석 → 스타일 노드 매칭

기존 서비스: 키워드 검색 or 카테고리 브라우징 or 채팅 (Daydream)
portal.ai: **사진 한 장 → 룩 분해 → 각 아이템의 스타일 DNA 추출 → 브랜드 감도 기반 매칭**

```
[유저 사진]
    ↓ GPT-4o-mini Vision
[룩 분해] → 각 아이템별 fit, fabric, color, detail, position
    ↓ Fashion Genome
[스타일 노드 매칭] → "C 미니멀_컨템퍼러리" → AMOMENTO, ANDERSSON BELL, ...
    ↓ 스코어링
[상품 추천] → 스타일 노드 부스트 + 키워드 매칭 + 브랜드 다양성
```

**이것은 캐치패션(메타서치)도, Daydream(채팅)도, 무신사(카테고리)도 하지 않는 방식이다.**

핵심 자산:
- **Fashion Genome DB**: 1,079 브랜드, 15개 스타일 노드(A-1 ~ K), 12종 감도 태그
- **스타일 노드 부스트 스코어링**: 단순 키워드 매칭이 아닌, 브랜드 감도 기반 추천
- **이미지 내 위치 좌표**: 각 아이템의 이미지 내 위치(%) → 인터랙티브 핫스팟 UI

### 3-2. 아시아 디자이너 브랜드 커버리지

| | Daydream | 무신사 | portal.ai |
|---|---|---|---|
| 서양 럭셔리 | ✅ 8,000 브랜드 | △ 일부 | △ CJ 어필리에이트 |
| 한국 디자이너 | ❌ 0개 | ✅ 폐쇄형 | ✅ 크롤링 |
| 일본 디자이너 | ❌ 유니클로만 | △ 일부 | ✅ 크롤링 확장 가능 |
| 아시아 인디 | ❌ | ❌ | ✅ 유일 |

현재 크롤링 중인 한국 브랜드 (일부):
OPEN YY, ANDERSSON BELL, SCULPTOR, aeae, CERRIC, AMOMENTO, Birrot, CFCL, Needles, Engineered Garments, Kapital, Eastlogue...

**Daydream이 $50M 시드를 받고도 못 하는 것을, portal.ai는 하고 있다.**

### 3-3. 하이브리드 소싱: 크롤링 + 어필리에이트

```
[아시아 브랜드] ← Cafe24 편집샵 크롤링 (9개, 확장 예정)
                  Fashion Genome 스타일 노드 매핑
                  → 차별화의 핵심. 다른 누구도 안 함.

[서양 브랜드]  ← CJ/AWIN 어필리에이트 피드
                  SSENSE, Farfetch, ASOS 등
                  → 수익화 채널. 커미션 5-13%.

[통합 검색]    ← 스타일 노드 기반 통합 스코어링
                  소스 무관하게 "이 사진과 어울리는 상품" 추천
```

이 구조가 캐치패션/Daydream과 다른 이유:
- 캐치패션: 서양만 있고, AI 분석 없고, 커미션만으로 수익 → 죽음
- Daydream: 서양만 있고, AI 있지만 아시아 브랜드 없음
- **portal.ai: 아시아 + 서양 + AI 분석 + 이중 수익구조**

## 4. 이중 수익구조 설계

### 4-1. 구독 (Primary Revenue)

캐치패션의 교훈: 커미션만으로는 죽는다. **AI 분석 자체에 가치를 매겨야 한다.**

| 플랜 | 월 요금 | AI 분석 | 상품 추천 | 부가 기능 |
|------|--------|---------|----------|----------|
| **Free** | 0원 | 하루 3건 | 기본 5개/아이템 | — |
| **Basic** | 4,900원 | 월 100건 | 상위 10개/아이템 | 분석 히스토리 저장 |
| **Pro** | 9,900원 | 월 500건 | 상위 15개 + 가격 알림 | 스타일 노드 상세 리포트 |
| **Premium** | 14,900원 | 무제한 | 전체 + 유사 룩 추천 | API 접근 (B2B) |

**왜 유저가 돈을 내는가:**
- 스타일 분석 자체의 가치 (옷 뭐 사야 할지 결정)
- 한국/아시아 디자이너 브랜드 검색 (다른 곳에 없음)
- 이미지 기반 → 텍스트 검색보다 정확한 매칭

**BEP 시뮬레이션:**

| 비용 항목 | 월 비용 |
|----------|--------|
| GPT-4o-mini Vision | ~6원/건 × 사용량 |
| Vercel Pro | ~20,000원 |
| Supabase | ~30,000원 |
| SerpApi (유료 전환 시) | ~130,000원 |
| **합계 (1,000명 기준)** | **~300,000원** |

```
BEP = 월 고정비 300,000원 ÷ 평균 구독료 7,000원 ≈ 약 43명
```

1,000명 구독 시: 매출 ~640만원, 비용 ~30만원 → **이익 ~610만원/월**
(구독 믹스: Free 60%, Basic 25%, Pro 10%, Premium 5% 가정)

### 4-2. 커미션 (Secondary Revenue)

| 소스 | 모델 | 예상 커미션 | 비고 |
|------|------|-----------|------|
| **CJ Affiliate** | CPA | SSENSE 7.5%, Farfetch 7% | 서양 럭셔리 |
| **AWIN** | CPA | ASOS 6% | 유럽 패션 |
| **Skimlinks** | CPA (자동) | 기본 커미션의 75% | 기존 크롤링 상품 래핑 |

**커미션 수익 시뮬레이션 (보수적):**

| 지표 | 수치 |
|------|------|
| 월간 상품 클릭 | 10,000건 (1,000 유저 × 10 클릭) |
| 전환율 | 2% (업계 평균) |
| 평균 주문가 | $200 (럭셔리/컨템포러리 평균) |
| 평균 커미션률 | 6% |
| **월 커미션 수익** | **10,000 × 2% × $200 × 6% = $2,400 (약 310만원)** |

### 4-3. 이중 구조 합산

| 수익원 | 1,000명 기준 | 비중 |
|--------|------------|------|
| 구독 | ~640만원 | 67% |
| 커미션 | ~310만원 | 33% |
| **합계** | **~950만원/월** | 100% |
| 비용 | ~30만원 | — |
| **순이익** | **~920만원/월** | — |

**캐치패션 vs portal.ai:**
- 캐치패션: 커미션 100% → 매출 40억이어도 영업손실 69억
- portal.ai: 구독 67% + 커미션 33% → 구독이 고정 수익 보장, 커미션은 보너스

## 5. 리스크 & 현실적 제약

### 리스크 1: 크롤링 법적 리스크

| 위험도 | 내용 |
|--------|------|
| **실제 사례** | 캐치패션 → 발란/트렌비/머스트잇 형사 고소 (정보통신망법 48조) |
| **대법원 판례** | 2021도1533 — 접근통제 우회 없는 크롤링은 무죄 |
| **현재 상태** | Cafe24 편집샵은 접근통제 없음 → 상대적 안전 |
| **위험 구간** | 무신사/29CM 등 대형 플랫폼 크롤링 시 고소 가능성 높음 |

**대응:**
- Cafe24 편집샵 크롤링 유지 (접근통제 없음 = 상대적 안전)
- 대형 플랫폼은 크롤링 대신 어필리에이트/파트너십으로 접근
- 크롤링 시 robots.txt 준수, rate limiting 적용
- 장기적으로 편집샵과 공식 데이터 제휴 추진

### 리스크 2: 커미션만으로는 수익 불가

캐치패션이 완벽하게 증명:
- 매출 40억 (대부분 커미션) vs 영업손실 69억
- 커미션 3-7%로는 마케팅비를 영원히 감당 못 함

**대응:** 구독이 메인, 커미션은 보조. 구독 없이 커미션만으로 가면 캐치패션 전철.

### 리스크 3: 통합 결제 부재

지그재그 교훈: "보여주고 외부로 보내기"만으로는 리텐션이 안 잡힘.

**대응:**
- 단기: AI 분석 가치 자체로 리텐션 (매번 새 사진 = 새 분석)
- 중기: 위시리스트/가격 알림으로 재방문 유도
- 장기: 통합 결제는 규모 확보 후 검토 (현 단계에서 과잉)

### 리스크 4: 승자독식 시장

무신사/카카오/신세계/네이버가 장악. 독립 플랫폼 생존률 극히 낮음.

**대응:**
- 정면 승부 안 함. "패션 쇼핑몰"이 아닌 "AI 패션 분석 도구"로 포지셔닝
- 무신사와 경쟁이 아닌, 무신사에서 사기 전에 쓰는 도구
- B2B 확장 가능성: 패션 브랜드/에디터에게 분석 API 제공

### 리스크 5: C-커머스 침공 (쉬인/테무)

중저가 패션이 초토화되고 있음.

**대응:**
- portal.ai는 중저가가 아닌 디자이너/컨템포러리 타겟
- 쉬인/테무 유저와 겹치지 않는 세그먼트
